Fisheries acoustics

I'm a member of the BAS fisheries acoustics GitHub group which publishes code and data for fisheries acoustic data processing. I've made some small contributions to echopy (a library of acoustic processing algorithms implemented in Python).

EchoJulia is a set of libraries for working with hydroacoustic data in Julia. With strong support for Simrad EK60 scientific echo sounders, it can be used to plot echograms and undertake various data analyses.

I'm interested in aliased seabed detection and removal. An initial algorithm is described in https://arxiv.org/abs/1904.10736. Implementations are available in Julia and Python.
